# 팩트 카드 — 2026-07-14 데일리

## 선정 토픽
OpenAI GPT-Live — 풀듀플렉스(전이중) 음성 모델. AI 음성 대화가 '순서 주고받기(턴테이킹)' 방식(반이중)에서 듣기와 말하기를 동시에 하는 방식(전이중)으로 넘어간 사건.

## 선정 사유
- 이번 주(7/8) 발표된 신선한 사건. 최근 발행 글이 법률(애플-오픈AI)·보안(JADEPUFFER)·정책(수출통제)·효율(토큰맥싱)에 몰려 있어, 제품·기술 본질을 다루는 글로 리듬 전환.
- '반이중 무전기 → 전이중 전화기'라는 공학 은유가 저자의 시그니처 모티프(1876년 벨의 전화기)와 정확히 맞물린다. 인문학적 IT기술론에 이상적인 소재.
- 팩트가 복수 출처(OpenAI 공식, TechCrunch, VentureBeat, TechTimes, Agora 측정 등)로 교차 확인.
- 비판적 균형 각도가 분명: '자연스러움 ≠ 효율/우월'. 전이중이 반이중보다 반드시 빠르지 않다(끼어들 때 오히려 지연). 저자의 '숫자의 맥락 재해석' 시그니처에 부합.

## 핵심 팩트 (모두 출처 재검증 대상)

- **발표일:** 2026년 7월 8일, OpenAI가 GPT-Live 공개. ChatGPT Voice를 구동하는 새 음성 모델 세대.
- **아키텍처:** 풀듀플렉스full-duplex — 듣기와 말하기를 **동시에** 수행. 기존 Advanced Voice Mode는 반이중half-duplex 턴테이킹(한 번에 한쪽만) 구조였다.
- **동작:** 초당 여러 번(“many times per second”) 말할지·계속 들을지·잠시 멈출지·“음mhmm”/“네yeah” 같은 맞장구backchannel를 넣을지·백그라운드 도구를 부를지를 결정.
- **변종 2종:** 
  - `GPT-Live-1` — Go·Plus·Pro 사용자 기본값.
  - `GPT-Live-1 mini` — 무료 사용자 기본값. **Advanced Voice Mode를 대체**.
  - 전 세계 ChatGPT 사용자에게 순차 롤아웃 시작.
- **백그라운드 모델:** 웹 검색·심층 추론·복잡한 작업이 필요하면 최신 프런티어 모델에 위임했다가 결과를 대화에 다시 가져온다. **출시 시점 백그라운드 모델은 GPT-5.5.**
- **API:** 개발자·기업은 API 접근 알림 신청 가능. “곧soon” 제공 예정(출시 시점엔 미개방).
- **Advanced Voice Mode:** OpenAI가 2024년 도입.
- **레이턴시 뉘앙스(중요, 균형 포인트):**
  - 구형 Realtime API의 총 왕복 지연은 약 1,700ms로 문서화(응답 첫 단어까지 약 1.5초 침묵).
  - Agora의 독립 측정: 전이중 GPT-Live는 상황에 따라 **오히려 지연이 늘 수 있다** — 사용자가 일부러 끼어들 때 Advanced(턴 기반 전작) 대비 약 498ms 회귀(느려짐). 전이중=무조건 빠름이 아니다. 자연스러움과 반응속도는 별개.

## 시점 규율 체크
- 발행일 2026-07-14 기준. GPT-Live는 7/8 발표된 **완료된 사건**(발표 완료, 롤아웃 진행 중). "예정"이 아니다.
- API는 아직 미개방 — "곧 제공 예정"으로 정확히 표기할 것.
- 백그라운드 모델은 GPT-5.5(출시 시점). GPT-5.6은 6/26 프리뷰·정부목록 게이팅 단계(ai-timeline 기준)이므로 GPT-Live 백엔드로 단정하지 말 것.
- ai-timeline상 미래 사건을 과거처럼 쓰지 않기.

## 평론 각도 제안
1. **무전기와 전화기 (도입 — 템플릿 A 역사대비형):** 반이중 무전기(“오버”라고 말해야 상대가 말한다) → 전이중 전화기(1876 벨). AI 음성이 무전기에서 전화기로 넘어간 순간.
2. **턴테이킹은 인간 대화의 근본:** 순서를 주고받는 것, 맞장구, 잠깐의 침묵 — 이것을 기계가 흉내낼 때 우리는 '사람 같다'고 느낀다. 그런데 그것은 이해인가, 이해의 연기인가(비판적 균형).
3. **숫자의 맥락 재해석(시그니처):** 전이중이 반이중보다 반드시 낫지 않다. 끼어들 때 오히려 498ms 느려진다. '자연스러움'과 '효율'은 다른 축. 더 인간적인 것이 더 나은 것은 아니다.
4. **한국적 맥락(마무리):** AI 콜센터·상담·실시간 통역. 한국어의 추임새 문화("네네", "그쵸")와 턴테이킹. 대화의 주도권을 누가 쥐는가 — 질문하는 인간 논지로 수렴.
