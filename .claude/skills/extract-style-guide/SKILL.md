# Extract Style Guide — 책에서 저자 스타일 추출

## 작업 순서

### 1단계: 샘플 읽기
책 파일이 큰 경우 전체를 한번에 읽지 말고 다음 5구간을 샘플링:
- 프롤로그 (0~200줄): 저자의 선언적 문장
- 1장 도입부 (200~500줄): 역사적 일화로 시작하는 전형
- 중반부 한 장 (1800~2200줄): 논지 전개
- 후반부 한 장 (2300~2600줄): 산업 분석
- 부록/가이드 (2700~3000줄): 실용적 어조

### 2단계: 관찰 기록

책에서 **반복 등장하는 패턴**을 수집한다:

- **도입부 스타터**: 어떤 소재로 시작하는가? (역사 인물, 일상 관찰, 특정 숫자, 비유적 장면)
- **어미 종결**: `~입니다 / ~이지요 / ~거든요 / ~습니다` 중 어떤 비율?
- **화자 거리**: "저는" 얼마나 자주 나타나는가? "여러분"은?
- **한국어-영어 병기**: `챗GPTChatGPT` 스타일? 괄호 병기?
- **강조 장치**: **볼드**, `> 인용구`, 번호 목록, 소제목 prefix(`▸`, `•`)
- **비유의 레퍼토리**: 과거 기술(원전, 전화기, 에니그마), 자연(연꽃, 나무), 신체(엄지손가락, 뇌)
- **반복 모티프**: "인문학적 IT기술론", "질문하는 인간", "공기처럼 흔해지는 AI"
- **마무리 패턴**: 독자에게 던지는 질문? 선언적 예언? 철학적 반문?

### 3단계: 금기 목록 추출

저자가 **쓰지 않는 것들**:
- 이모지 (거의 없음)
- 해시태그
- 과장된 감탄사 ("대박!", "헐!")
- 줄임말 (ㅎㅎ, ㅋㅋ)
- 영어 단독 헤드라인
- "카피라이터스러운" 홍보 문구

### 4단계: 한 꼭지 해부

책에서 대표적인 한 장을 골라 구조를 태깅:
```
[도입부: 역사 일화 - 러다이트]
[전환: 현재 AI 시대와의 공명]
[본문 소제목 1: ▸ 러다이트의 진짜 공포]
[팩트 블록: 연도, 인물, 수치]
[비판적 균형: 양측 견해]
[마무리: "우리는 어떻게 할 것인가" 질문형]
```

### 5단계: 쓰기 템플릿 제공

ghostwriter가 바로 참조할 수 있는 5가지 도입부 템플릿:

1. **역사 대비형**: "[N년 전의 사건], 그로부터 N년이 지난 지금 [유사 상황]"
2. **일상 관찰형**: "[일상 장면 묘사]. 이런 풍경이 어색하지 않은 시대가 되었습니다."
3. **숫자 충격형**: "[놀라운 숫자]. 이것이 [사건]의 규모입니다."
4. **인물 일화형**: "[특정 인물의 일화]. 그는 [AI와 연결되는 행동]을 했습니다."
5. **질문 도발형**: "[철학적·도발적 질문]? 이 질문이 2026년에 다시 울리고 있습니다."
