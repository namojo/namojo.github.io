GitHub Personal Access Token 설정

GitHub에 직접 저장하려면 Personal Access Token이 필요합니다.

토큰 생성 방법:

  1. GitHub Token 생성 페이지로 이동
  2. Note에 "Jekyll Blog Editor" 입력
  3. Expiration은 원하는 기간 선택
  4. 다음 권한 체크:
    • ✅ repo (Full control of private repositories)
  5. "Generate token" 클릭
  6. 생성된 토큰을 아래에 붙여넣기

⚠️ 토큰은 브라우저에만 저장되며, 다른 곳으로 전송되지 않습니다.
